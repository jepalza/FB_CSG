# FB_CSG
FreeBasic Creacion de objetos STL mediante CSG

Partiendo de los fuentes originales en C de esta página:
http://c-csg.com/
He modificado ligeramente los ficheros para permitir desde FreeBasic enviar comandos a la DLL generada.

Sirve para la creación de objetos STL fusionados, restados o divididos y su posterior guardado para su empleo en impresoras 3D.
